## Overview

This repository contains all code, results, and figures supporting our MSc thesis. The thesis investigates whether synthetic fraud data generated by the Synthetic Data Vault (SDV) library can substitute for or improve upon real transaction data when training credit card fraud detection models. Three experiments are conducted across four classical ML classifiers and four deep learning architectures, with all results reported as mean ± standard deviation across five random seeds.

**Key findings:**
- CTGAN synthetic-only training achieves 81.36% F1 — only 2.75 percentage points below the real-data baseline (84.11%)
- CTGAN augmentation at 5% fraud ratio marginally surpasses the real-data baseline (84.46% F1)
- TabPFN improves from 16.68% to 81.75% F1 through 50% synthetic augmentation
- No single imbalance handling technique is universally optimal across all scenarios

## Experimental Framework

| Experiment | Training Data | Purpose | RQ Addressed |
|---|---|---|---|
| Experiment 1 | Real data (226,980 rows, 378 fraud) | Real data baseline | RQ1 |
| Experiment 2 | CTGAN synthetic only (300,000 rows) | Synthetic-to-real generalisation | RQ2 |
| Experiment 3 | Real + CTGAN fraud augmentation (5 ratios) | Augmentation sensitivity | RQ2, RQ3 |

**Models evaluated:** Logistic Regression, Random Forest, XGBoost, LightGBM, TabNet, Tabular ResNet, VAE, TabPFN

**Imbalance handling techniques:** No Handling, SMOTE, ADASYN, Cost-Sensitive Learning, Random Undersampling

**Augmentation ratios (Experiment 3):** 1%, 5%, 10%, 25%, 50%

All results reported as mean ± standard deviation across five random seeds: {42, 123, 456, 789, 1234}

## How to Run

**1. Clone the repository**
```bash
git clone [INSERT GITHUB URL]
cd [repo-name]
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Download the dataset**

The Kaggle ULB Credit Card Fraud Detection dataset is **not included** in this repository due to licensing. Download it from:  
👉 https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Place the downloaded `creditcard.csv` file in a `/data` folder at the root of the repository.

**4. Run the scripts in order**
01_EDA.py                        → Exploratory data analysis
03a/03b/03c                      → Train the three synthesisers
04_synthesiser_quality_check.py  → Evaluate synthesiser quality
05_generate_best_fraud_250k.py   → Generate fraud-only synthetic pool
02_experiment1_real_data.py      → Run Experiment 1
06_experiment2_synthetic_only.py → Run Experiment 2
07_experiment3_augmentation.py   → Run Experiment 3
08_experiment4_deep_learning.py  → Run deep learning models
10_merge_all_results.py          → Merge all results
09_generate_figures.py           → Generate all figures

> ⚠️ **Note:** Training the CTGAN synthesiser takes approximately 9 hours on CPU. Pre-trained synthesiser outputs are included in the repository where file size permits, so it can be skipped straight to the experiment scripts
